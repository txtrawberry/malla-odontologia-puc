# Malla Interactiva — Odontología PUC

Aplicación web interactiva para llevar el seguimiento de tu avance en la carrera de **Odontología en la Pontificia Universidad Católica de Chile (PUC)**.

## Características
- **Desbloqueo automático de ramos:** Al marcar un curso como aprobado (haciendo clic sobre él), se evalúan los requisitos y se desbloquean automáticamente los siguientes ramos de la malla.
- **Contador de créditos y ramos:** Visualiza en tiempo real cuántos créditos y ramos llevas aprobados.
- **Persistencia local:** Tus datos se guardan en el navegador (`localStorage`) para que no pierdas tu avance al recargar.
- **Diseño limpio y responsivo:** Adaptado para celulares, tablets y computadoras.

---

### ¿Cómo subirlo a GitHub Pages en 2 minutos?
1. Crea un repositorio nuevo en GitHub (por ejemplo, `malla-odontologia-puc`).
2. Sube estos tres archivos (`index.html`, `style.css`, `script.js`) a la rama principal (`main`).
3. Ve a **Settings** > **Pages** en tu repositorio de GitHub.
4. En **Build and deployment**, selecciona la rama `main` y carpeta `/ (root)`, luego da clic en **Save**.
5. ¡Listo! GitHub te dará un enlace público donde podrás ver y usar tu malla interactiva desde cualquier dispositivo.
